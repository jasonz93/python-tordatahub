PyDatahub: Datahub Python SDK
=====================================

`PyDatahub <https://github.com/aliyun/aliyun-datahub-sdk-python>`_ 是Datahub的Python版本的SDK，它对Datahub服务提供的各个RESTful API接口进行了封装，提供了简单方便的Python编程接口。有关Datahub服务的详细介绍请参见 `阿里云官网介绍 <https://help.aliyun.com/document_detail/47439.html?spm=5176.doc47440.6.578.UUlqBz>`_ 。

Requirements:
  * setuptools (>=3.0)
  * requests (>=2.4.0)
  * simplejson (>=3.3.0)
